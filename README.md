## рҹ§  ЩҶШёШұШ© Ш№Ш§Щ…Ш©

**RoboOS** ЩҶШёШ§Щ… ШӘШҙШәЩҠЩ„ Щ…ШҜЩ…Ш¬ Щ…ШөЩ…Щ… Щ…ЩҶ Ш§Щ„ШөЩҒШұ Щ„ЩҲШӯШҜШ§ШӘ Ш§Щ„ШӘШӯЩғЩ… Ш§Щ„ШҜЩӮЩҠЩӮШ© ARM Cortex-M4ШҢ ЩҠШ¬Щ…Ш№ ШЁЩҠЩҶ:

- рҹ”© **ЩҶЩҲШ§Ш© RTOS ШӯЩӮЩҠЩӮЩҠШ©** Щ…ЩғШӘЩҲШЁШ© ШЁШ§Щ„ШЈШіЩ…ШЁЩ„ЩҠ (context switch, syscalls, boot)
- вҡҷпёҸ **Ш·ШЁЩӮШ© HAL** Щ„ШӘШ¬ШұЩҠШҜ Ш§Щ„ЩҮШ§ШұШҜЩҲЩҠШұ Ш№ЩҶ ШЁЩӮЩҠШ© Ш§Щ„ЩҶШёШ§Щ…
- рҹ–ҘпёҸ **Щ…ШӯШ§ЩғЩҠ ЩғШ§Щ…Щ„** ЩҠЩҸШҙШәЩ‘Щ„ Ш§Щ„ЩҶШёШ§Щ… Ш№Щ„Щү Ш§Щ„ЩғЩ…ШЁЩҠЩҲШӘШұ ШЁШҜЩҲЩҶ MCU
- рҹ”Ө **Щ„ШәШ© ШЁШұЩ…Ш¬Ш© Щ…Ш®ШөШөШ©** Щ„Щ„ШӘШӯЩғЩ… ШЁШ§Щ„ШұЩҲШЁЩҲШӘ (Lexer вҶ’ Parser вҶ’ Executor)
- рҹӨ– **Robot APIs** Ш№Ш§Щ„ЩҠШ© Ш§Щ„Щ…ШіШӘЩҲЩү Щ„Щ„ШӘШӯЩғЩ… ШЁШ§Щ„Щ…ШӯШұЩғШ§ШӘ ЩҲШ§Щ„Щ…ШіШӘШҙШ№ШұШ§ШӘ ЩҲLED
